# Évaluation immobilière — Méthodes et pratiques

**Source :** Pratiques professionnelles géomètre-expert + experts immobiliers ; Charte de l'expertise en évaluation immobilière (mise à jour 2024) ; norme NF EN 16310 (Services d'ingénierie pour évaluations immobilières) ; SCPI / pratiques foncières.

## Cadre légal

### Profession et qualifications

L'**évaluation immobilière** peut être réalisée par :
- **Expert immobilier** inscrit sur la liste de la **Cour d'appel** (expertise judiciaire)
- **Géomètre-expert** dans le cadre de sa mission technique
- **Notaire** dans le cadre de sa mission de conseil
- **Agent immobilier** (estimation commerciale, pas une expertise)

### Cas où une expertise est obligatoire

- **Succession** (déclaration au notaire)
- **Donation** ou partage
- **Vente** avec litige (expertise judiciaire)
- **Expropriation** (juge de l'expropriation)
- **Impôts** : IFI, plus-value (justification de valeur)
- **Comptabilité** : bilan, immobilisation
- **Assurance** : sinistre

## Trois méthodes principales

### 1. Méthode par **comparaison**

#### Principe

Comparer le bien à évaluer à des **biens similaires** récemment vendus dans la zone.

#### Étapes

1. **Recensement** de transactions comparables :
   - Source : **Notaires de France** (perval.fr), bases prix immobilier (BIEN à Paris, MIN à province)
   - Période : 6-12 mois max (marché évolue)
   - Zone : rayon 1-5 km selon densité
2. **Sélection** des comparables (3-5 biens minimum)
3. **Pondération** selon écarts :
   - Surface (€/m² vs total)
   - État (neuf, à rénover)
   - Étage (1ᵉʳ vs 5ᵉ)
   - Vue, exposition
   - Annexes (parking, cave, terrasse)
4. **Calcul** de la valeur ajustée

#### Avantages

- Méthode **dominante** pour les biens courants (logement, parking, commerce)
- Refleter le marché réel
- Facilement vérifiable

#### Limites

- Difficile pour biens **atypiques** (manoirs, biens industriels)
- Nécessite des transactions **récentes** comparables
- Sensible aux **variations** de marché

### 2. Méthode par **capitalisation** (rendement)

#### Principe

Valeur = Revenu locatif annuel net / Taux de capitalisation

```
Valeur = R_net / T_cap
```

#### Application

- Biens **locatifs** (locaux d'habitation loués, bureaux, commerce)
- Investissement (point de vue investisseur)

#### Calcul du revenu net

```
R_brut = Loyer annuel HT
- Charges non récupérables (~ 5-15 % du loyer)
- Taxe foncière
- Vacance locative prévue (~ 5-10 %)
- Frais de gestion (~ 5-8 % si syndic ou agence)
= R_net
```

#### Taux de capitalisation indicatifs

| Type de bien | Taux de capitalisation |
|---|---|
| Logement Paris haut de gamme | 2,5-3,5 % |
| Logement Paris standard | 3-4 % |
| Logement IDF couronne | 4-5 % |
| Logement province (grande ville) | 4-6 % |
| Logement province (petite ville) | 5-7 % |
| Logement étudiant | 5-7 % |
| Bureau Paris CBD | 3-4 % |
| Bureau Paris hors CBD | 4-5 % |
| Bureau province | 6-8 % |
| Commerce centre-ville | 5-7 % |
| Local industriel | 7-9 % |
| Entrepôt logistique | 6-8 % |

#### Exemple

Appartement loué 12 000 €/an HT à Lyon :
- Charges non récupérables : 800 €
- Taxe foncière : 1 200 €
- Vacance prévue : 600 €
- Frais de gestion : 720 €
- **R_net** : 12 000 - 3 320 = 8 680 €
- Taux de capitalisation (Lyon) : 5 %
- **Valeur** : 8 680 / 0,05 = **173 600 €**

#### Avantages

- Reflète le **rendement** réel
- Utile pour **investisseurs**

#### Limites

- Sensible au taux de capitalisation choisi
- Dépend de la qualité du locataire et du bail
- Ne convient pas aux biens vacants ou non locatifs

### 3. Méthode par le **coût de remplacement**

#### Principe

Valeur = Coût de reconstruction à neuf - Vétusté + Valeur foncière

```
Valeur = (C_neuf - V) + V_fonciere
```

#### Application

- Biens **uniques** (industriels, ouvrages spéciaux)
- Quand la comparaison est difficile
- Cas où l'absence de marché ne permet pas la capitalisation

#### Calcul

- **C_neuf** : coût de reconstruction à neuf du bâtiment (€/m² × surface)
- **V** : abattement pour vétusté (5-50 % selon âge et état)
- **V_fonciere** : valeur du terrain (par comparaison)

#### Coûts de reconstruction (références)

| Type | Coût reconstruction (€/m² SDP) |
|---|---|
| Logement collectif standard | 1 800-2 400 |
| Maison individuelle standard | 1 500-2 200 |
| Bureau tertiaire | 1 800-2 500 |
| Commerce | 1 500-2 200 |
| Industriel (atelier) | 800-1 500 |
| Hôtel | 2 500-4 000 |

#### Abattement vétusté

| Âge | Abattement |
|---|---|
| < 5 ans | 5-10 % |
| 5-15 ans | 10-20 % |
| 15-30 ans | 20-35 % |
| 30-50 ans | 35-50 % |
| > 50 ans | 50-70 % (selon entretien) |

#### Limites

- **Méthode théorique** — pas toujours alignée avec le marché
- **Foncier** difficile à isoler
- Utilité limitée pour biens courants

## Démarche d'évaluation type

### Étape 1 — Mission et contexte

- **Objet** de l'évaluation (vente, succession, assurance, IFI, etc.)
- **Date** d'effet (valeur au jour de…)
- **Périmètre** : un bien ou un ensemble ?
- **Type de valeur** :
  - **Valeur vénale** : prix probable de vente
  - **Valeur d'usage** : pour le propriétaire actuel
  - **Valeur de remplacement** : reconstruction
  - **Valeur locative**
  - **Valeur d'expertise judiciaire**

### Étape 2 — Étude documentaire

- **Acte de propriété** + descriptif
- **Plan cadastral**
- **Titre de propriété**
- **Diagnostics techniques** (DPE, amiante, plomb, etc.)
- **Loyers** (baux en cours)
- **Charges** de copropriété
- **Permis de construire** ou DP éventuels

### Étape 3 — Visite et levé

- **Visite** du bien (intérieur + extérieur)
- **Surface** mesurée (Carrez si copropriété)
- **État** général :
  - Bon entretien / Vétusté moyenne / Mauvais état
  - Travaux à prévoir (chiffrage estimé)
- **Photos** datées
- **Environnement** (quartier, transports, écoles)

### Étape 4 — Recherches marché

- **Comparables** (Perval, agences locales, observatoires immobiliers)
- **Indices** : indice notaires, indice INSEE
- **Tendance** : marché en hausse / baisse / stagnation
- **Spécificités** locales (rareté, demande forte)

### Étape 5 — Application des méthodes

- **Méthode comparative** (souvent prioritaire pour logement)
- **Méthode capitalisation** (si bien locatif)
- **Méthode coût de remplacement** (cas particuliers)
- **Croisement** : confrontation des résultats

### Étape 6 — Synthèse

#### Tableau récapitulatif

| Méthode | Valeur |
|---|---|
| Comparative | _ € |
| Capitalisation | _ € |
| Coût de remplacement | _ € |
| **Valeur retenue** | **_ €** |

#### Justification

- Pondération entre méthodes (selon contexte)
- Élément retenu (médiane, moyenne, méthode dominante)
- **Fourchette** de prix (basse / haute) plutôt que valeur unique
- **Marge d'incertitude** (± 5-15 % selon contexte)

### Étape 7 — Rapport

#### Structure

```
RAPPORT D'EXPERTISE IMMOBILIÈRE

1. MISSION ET CONTEXTE
   - Mandant
   - Objet
   - Date d'effet

2. IDENTIFICATION DU BIEN
   - Adresse
   - Références cadastrales
   - Description
   - Surface (Carrez si applicable)

3. ÉTUDE JURIDIQUE
   - Statut (pleine propriété, démembrement, indivision)
   - Charges et servitudes
   - Litiges éventuels

4. ÉTUDE TECHNIQUE
   - État général
   - Travaux à prévoir
   - Diagnostics techniques

5. ÉTUDE ÉCONOMIQUE
   - Marché local
   - Comparables
   - Tendances

6. ÉVALUATION
   - Méthodes appliquées
   - Calculs détaillés
   - Synthèse

7. CONCLUSION
   - Valeur vénale retenue
   - Fourchette de prix
   - Validité de l'expertise

ANNEXES :
- Plans, photos, tableaux comparables, etc.
```

## Cas particuliers

### Évaluation pour IFI (Impôt sur la Fortune Immobilière)

- Valeur **vénale au 1ᵉʳ janvier** de l'année
- Justifiable en cas de contrôle DGFiP
- Méthode comparative + capitalisation

### Évaluation pour succession

- Valeur **vénale au jour du décès**
- Pour calcul des droits de succession
- Préférable d'évaluer **bas** (moins de droits) mais justifiable

### Évaluation pour expropriation

- **Indemnité d'expropriation** = valeur vénale + indemnités annexes
- Procédure devant le **juge de l'expropriation**
- Expertise judiciaire désignée

### Évaluation pour partage / divorce

- Valeur **vénale au jour du partage**
- Souvent **expertise contradictoire** (chaque partie peut désigner un expert)
- Médiation possible

### Évaluation d'un bien commercial (fonds de commerce)

- Méthode spécifique (rendement, comparaison, coefficient)
- Intégration des éléments incorporels (droit au bail, clientèle, enseigne)

## Marché immobilier — sources

### Bases de données

- **Perval** (Notaires de France) : transactions notariales
- **BIEN** (Paris et IDF) : transactions
- **MIN** (province) : transactions
- **DVF** (Données Valeurs Foncières — DGFiP) : transactions publiques
- **MeilleursAgents, SeLoger, Logic-Immo** : prix d'annonces (biais à la hausse)

### Indices

- **Indice notaires-INSEE** logement ancien : mise à jour trimestrielle
- **Indice IRL** (loyers) : mise à jour annuelle
- **Indice ILC** (loyers commerciaux)
- **ILAT** (loyers tertiaires)

## Coût et délai

### Coût indicatif

| Mission | Coût HT |
|---|---|
| Évaluation MI standard | 800-2 000 € |
| Évaluation appartement | 600-1 500 € |
| Évaluation logement collectif (immeuble) | 2 500-8 000 € |
| Évaluation bureau / commerce | 1 500-5 000 € |
| Expertise judiciaire | 3 000-15 000 € |
| Évaluation pour IFI (forfait simple) | 500-1 200 € |

### Délai

- **Étude + visite** : 1-3 jours
- **Recherches** : 1-2 semaines
- **Rapport finalisé** : 2-4 semaines

## Synthèse — bonnes pratiques GE

1. **Mission claire** définie au démarrage (objet, périmètre, valeur recherchée)
2. **Étude documentaire** approfondie (titres, plans, diagnostics)
3. **Visite contradictoire** avec le propriétaire / mandant
4. **Méthode comparative** prioritaire pour logements courants
5. **Capitalisation** pour biens locatifs
6. **Comparables** pertinents (récents, locaux, similaires)
7. **Fourchette de prix** plutôt que valeur unique (5-15 % d'incertitude)
8. **Rapport détaillé** justifiant les choix méthodologiques
9. **Indépendance** vis-à-vis des intérêts du mandant (jamais favoriser une partie)
10. **RC pro** spécifique évaluation
11. **Mention finale** : valeur valable à date X, conditions de marché
