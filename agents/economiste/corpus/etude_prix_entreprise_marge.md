# Étude de prix entreprise — Marge, coefficient et négociation

**Source :** Pratiques professionnelles UNTEC + BTP — études de prix entreprise ; Bordereaux Le Moniteur / BatiPrix ; doctrine comptable BTP (PCG) ; pratiques de Compagnonnage et FFB.

## Distinction MOE / Entreprise

| Côté MOE | Côté Entreprise |
|---|---|
| **Estimation prévisionnelle** | **Étude de prix opérationnelle** |
| Bordereaux moyens (BatiPrix, UNTEC) | Coûts réels + ratios internes |
| Pas de bénéfice ni d'aléas spécifiques entreprise | Marge à intégrer |
| Objectif : enveloppe MOA | Objectif : devis pour client + viabilité |

## Décomposition d'un Prix Unitaire — méthode UNTEC

### Formule générale

```
PU = DS + FC + FG + BA
```

- **DS** : Déboursé Sec — matière + main d'œuvre + matériel d'exécution
- **FC** : Frais de Chantier — installations, gardiennage, base vie, échafaudages, sécurité collective
- **FG** : Frais Généraux d'entreprise — direction, comptabilité, assurances, formation
- **BA** : Bénéfice et Aléas — marge entreprise + aléas

### Pondération courante

- **FC** : 8 à 12 % du DS
- **FG** : 7 à 14 % du (DS+FC) selon taille entreprise
- **BA** : 5 à 10 % du (DS+FC+FG)

### Coefficient de vente

```
K_vente = PU / DS
```

#### Exemple

DS = 100 € ; FC 10 % = 10 € ; FG 10 % = 11 € ; BA 7 % = 8,47 €
PU = 100 + 10 + 11 + 8,47 = **129,47 €**
K = 1,29 (soit coefficient 1,29 ou +29 %)

### Coefficients de vente type par taille d'entreprise

| Taille | Coefficient de vente K |
|---|---|
| Artisan / petite entreprise < 5 salariés | 1,20 - 1,35 |
| PME 5-30 salariés | 1,30 - 1,55 |
| Entreprise moyenne 30-100 salariés | 1,40 - 1,65 |
| Grande entreprise > 100 salariés | 1,50 - 1,80 |
| Travaux publics | 1,45 - 1,75 |
| Grands projets / spécialités | 1,60 - 2,00 |

## Déboursé Sec — composition

### A. Matière (matériaux)

- **Prix d'achat** négocié auprès des fournisseurs (souvent -20 à -45 % par rapport au prix catalogue)
- **Coefficient de pertes** : 5-15 % selon ouvrage (chutes, casse, refoulé)
- **Transport et manutention** : 3-8 % du prix achat
- **Stockage et gestion** : 1-3 %

#### Exemple — béton C25/30

- Achat usine BPE : 105 €/m³
- Pertes (chutes coffrage, refoulé) : +5 % = 110 €/m³
- Transport (camion toupie) : +15 €/m³ = 125 €/m³
- **Coût matière** : 125 €/m³

### B. Main d'œuvre

- **Salaire brut** + cotisations sociales + CIBTP + OPPBTP + Prévoyance + CSG/CRDS
- Coefficient de charges sociales : **environ 50-60 %** du brut
- **Productivité** mesurée en h / unité d'ouvrage

#### Exemple — pose de mur parpaing 20

- Productivité : 1 h/m² (ouvrier qualifié)
- Salaire brut horaire : 18 €/h
- Charges sociales (55 %) : +9,90 €/h = 27,90 €/h
- **Coût MO** par m² = 27,90 € (sans aide)
- Avec aide-maçon (productivité 1,5 h/m² combinée) : ~ 25 €/m²

### C. Matériel d'exécution

#### Petit matériel

- Brouettes, étais, pelles, outils manuels
- **Forfait** par chantier ou %

#### Gros matériel

- Grue, échafaudage, banche, coffrage, vibreur
- **Location** journalière + amortissement
- **Frais de transport** sur chantier

#### Exemple — Banches de coffrage

- Location 50 €/jour
- Productivité voile 20 cm : 2 m²/jour-banche
- **Coût** : 25 €/m² de voile

### D. Sous-traitance interne (si applicable)

- Sous-traitance technique (terrassement, électricité)
- À intégrer en DS si gérée comme un déboursé direct

## Frais de Chantier (FC) — détail

### Composantes

- **Installation et repli de chantier** : clôture, panneaux, base vie, sanitaires
- **Gardiennage** : surveillance nocturne, week-ends
- **Échafaudages** (si > 6 m d'élévation)
- **Sécurité collective** : garde-corps, filets, rampes
- **Équipement de protection collective** : extincteurs, balisage
- **Énergie chantier** : électricité, eau chantier
- **Frais administratifs de chantier** : conducteur de travaux, chef de chantier, comptabilité chantier
- **Assurance Tous Risques Chantier** (TRC) si à charge entreprise

### Évaluation

#### Par % du DS (méthode rapide)

8 à 12 % du DS selon taille du chantier (gros chantiers > 1 M€ : plus bas — 6-8 %)

#### Par montant fixe (méthode rigoureuse)

Décomposition des éléments avec amortissement / mois de chantier :
- Installation + repli : forfait
- Gardiennage : x €/mois × durée
- Conducteur de travaux : y €/mois × durée
- Chef de chantier : z €/mois × durée
- Etc.

## Frais Généraux d'entreprise (FG) — détail

### Composantes

- **Direction** (salaire dirigeant + charges)
- **Service commercial** (commerciaux, devis)
- **Comptabilité, gestion** (interne + expert-comptable)
- **Assurances RC pro + RC Décennale**
- **Formation** (CPF, plan formation)
- **Frais financiers** (intérêts emprunts, charges bancaires)
- **Cotisations professionnelles** (FFB, CAPEB, syndicats)
- **Frais de bureaux** (loyer siège, énergies, télécoms)
- **Véhicules** (location, carburant, entretien)
- **Formation continue**, **médecine du travail**
- **Qualifications RGE, Qualibat**

### Évaluation

```
Taux FG = Frais Généraux annuels / Chiffre d'affaires annuel
```

### Ratios par taille d'entreprise

| Taille | Taux FG du CA |
|---|---|
| Artisan < 5 salariés | 7-12 % |
| PME 5-30 salariés | 9-14 % |
| Entreprise 30-100 salariés | 10-15 % |
| Grande entreprise > 100 salariés | 11-17 % |

→ Une entreprise plus grosse a des **FG plus lourds** (administration, structure).

## Bénéfice et Aléas (BA) — détail

### Bénéfice

- **Marge nette** souhaitée par l'entreprise
- Variable selon **risque** du chantier, **concurrence**, **stratégie commerciale**
- Cible **résultat net** : 3-8 % du CA (BTP — secteur peu rentable)

### Aléas

- **Risques techniques** non identifiés
- **Sujétions** non prévues (sol, météo, accès)
- **Évolutions** matières premières en cours d'exécution
- **Imprévus** organisationnels

### Évaluation

| Type de chantier | Taux BA |
|---|---|
| Chantier simple, court | 5-7 % |
| Chantier standard | 7-10 % |
| Chantier complexe, risqué | 10-15 % |
| Chantier exceptionnel (HQE++, ITGH) | 12-20 % |

## Méthode d'étude de prix complète

### Étape 1 — Réception du DCE

- Lecture **complète** : CCTP, DPGF, plans
- **Questions** au MOE par mail/voie DCE avant date limite
- **Validation** des hypothèses (sol, accès, programme, planning)

### Étape 2 — Métré et quantitatif

- **Métré** précis par poste
- **Validation** des quantités du DPGF (rarement justes)
- **Liste** des postes oubliés ou doublons

### Étape 3 — Consultation fournisseurs

- **Demande de prix** aux fournisseurs principaux pour les postes majeurs
- **Validation** des délais d'approvisionnement
- **Négociation** des prix selon volume

### Étape 4 — Étude de prix poste par poste

Pour chaque poste DPGF :
- DS = Matière + MO + Matériel + (Sous-traitance si applicable)
- FC % à appliquer
- FG % à appliquer
- BA % à appliquer
- PU final

### Étape 5 — Synthèse et négociation interne

- **Total HT** du chiffrage
- **Marge globale** calculée
- **Validation** par la direction (selon montant et risque)
- **Réajustement** stratégique si nécessaire (alignement marché, geste commercial)

### Étape 6 — Mémoire technique

Joint à l'offre :
- Méthodologie de chantier
- Planning prévisionnel
- Moyens humains et matériels
- Démarche qualité et sécurité
- Démarche environnementale
- Références similaires
- Notation : 30-50 % de l'évaluation finale (souvent négligé à tort)

### Étape 7 — Remise et négociation client

- **Devis détaillé** transmis au MOA / MOE
- **Présentation** orale si demandée
- **Négociation** : justification du prix, comparaison vs concurrents, modulation possible (variantes, options)

## Stratégies de prix et négociation

### Stratégie d'offre

| Stratégie | Quand l'utiliser | Risque |
|---|---|---|
| **Prix bas** (BA = 3-5 %) | Concurrence agressive, besoin remplir carnet | Marge faible, risque tensions |
| **Prix standard** (BA = 7-10 %) | Marché équilibré | Standard |
| **Prix premium** (BA = 12-18 %) | Spécialité, peu de concurrents | Possibilité d'être éliminé |
| **Dumping** (BA < 0) | Acquérir un client stratégique | Très risqué, OAB possible |

### Détection des Offres Anormalement Basses (OAB)

Si le candidat propose un prix manifestement bas :
- Le MOA peut **demander des justifications** (CCP L.2152-5)
- L'entreprise doit prouver la viabilité (procédés innovants, conditions exceptionnelles)
- À défaut : rejet de l'offre

### Critères de notation des offres (RC)

| Critère | Pondération courante |
|---|---|
| Prix | 40-60 % |
| Valeur technique (mémoire) | 30-50 % |
| Délai | 5-15 % |
| Environnement / RSE | 5-15 % |
| Insertion sociale | 5-10 % |

→ Un mémoire technique solide peut compenser un prix légèrement supérieur.

## Pièges et risques courants

| Piège | Conséquence |
|---|---|
| **Oubli d'un poste** | Perte sèche en exécution (sauf TS demandés au MOA — rarement obtenus) |
| **Sous-évaluation quantités** | Perte sur les déboursés directs |
| **Sous-évaluation MO** | Perte sur la productivité |
| **FG sous-estimés** | Perte cachée à long terme |
| **BA insuffisant** | Pas de marge pour imprévus |
| **Marché à BPU avec quantités sous-estimées** | Perte si quantités réelles < prévues, sans recours |
| **Ignorer la révision de prix** | Perte sur l'inflation pendant exécution |
| **Pas de mémoire technique solide** | Notation faible, perte du marché |

## Synthèse — bonnes pratiques entreprise

1. **Métré rigoureux** : ne pas se contenter du DPGF MOE.
2. **Consultations fournisseurs** : valeurs réelles, pas catalogue.
3. **Productivité MO mesurée** : suivi entreprise interne, pas standard générique.
4. **FG calculés annuellement** : extrait de l'année précédente du bilan.
5. **BA modulé** : selon risque et concurrence.
6. **Mémoire technique soigné** : facteur clé d'attribution.
7. **Validation interne** : direction + commercial avant remise.
8. **Suivi exécution** : comparaison budget vs réel en temps réel.
9. **Capitalisation** : retour d'expérience par chantier, mise à jour des ratios.
